# ハウス自動開閉機（Greenhouse Opener）制御基板製作用Kicadファイル

## 概要

Arduino を使った温室ハウスの自動開閉システムです。
現在開発中のオプション（WiFi通信、ハウス20棟まで対応可能、VPNサービスで外出先でも温度監視、開閉操作）
に対応しています。

## 構成
*Kicad-MainBoard*
  * メイン制御基板（モーター制御・センサー）
    
* Kicad-LCD*
  * 表示用LCD基板

## 特徴

* ローカルで完結（Wi-Fiなしでも動作可能）
* 農業現場で実運用中

## ファイルについて

* `.kicad_sch` → 回路図
* `.kicad_pcb` → 基板レイアウト
* `.kicad_pro` → プロジェクトファイル
* `Gerber.zip` → 基板発注用データ

KiCad バージョン 9.0 で作成しています。
